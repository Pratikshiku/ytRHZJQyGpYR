### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 个人中心（修改头像、修改个人信息、修改密码）
4. 航班收藏
5. 机票预订
6. 首页
7. 论坛
8. 评论留言
9. 航班信息
10. 民航信息
11. 在线客服

**管理员功能：**
1. 登录
2. 首页
3. 个人中心、修改密码
4. 管理员管理
5. 用户管理
6. 航班信息管理
7. 航班收藏管理
8. 机票预定管理
9. 客服聊天管理
10. 基础数据管理（航班类型、新闻类型）
11. 论坛管理
12. 民航新闻管理
13. 轮播图管理
